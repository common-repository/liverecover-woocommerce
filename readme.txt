== Description ==

LiveRecover provides a human-powered, text message-based abandoned cart recovery tool for WooCommerce stores.

The tool uses contact information from the checkout process to follow up with customers manually. This process enables LiveRecover to answer customer questions in real-time to help convert the abandoned cart into a sale.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `liverecover-woocommerce-plugin` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the LiveRecover API key in Woocommerce > Settings > Integration > LiveRecover WooCommerce Plugin
